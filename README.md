## Language Support

The app supports both English and Spanish. To change the language:

1. Go to the **Settings** tab (gear icon)
2. Select your preferred language under the **Language** section
3. The app will instantly update to your selected language

### Pre-loaded Exercises

When you first install the app, it comes pre-loaded with 29 exercises in your selected language:

- 14 gym exercises (bench press, squats, deadlift, etc.)
- 7 cardio exercises (running, cycling, rowing, etc.)
- 8 abs exercises (crunches, plank, leg raises, etc.)

## Getting Started

### Prerequisites

- Node.js
- Expo CLI
- React Native development environment

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

Then use the Expo Go app to scan the QR code, or run on an emulator:

```bash
npm run android  # For Android
npm run ios      # For iOS
```
